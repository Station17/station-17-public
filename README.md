## 🏙 О проекте

**Station-17** — это амбициозный ролевой проект на движке **Robust Toolbox** (C#). Мы объединили глубокую симуляцию **Space Station 14** с гнетущей атмосферой тотального контроля вселенной **Half-Life 2**.

Наш сервер ориентирован на жесткое социальное взаимодействие, бюрократию Альянса и динамическое изменение ролей прямо в ходе раунда.

---

## 🛰 Ключевые системы и механики

* **Единая База Данных:** Все граждане занесены в систему. Ваш статус, CID-карта и история лояльности сохраняются между раундами.
* **Динамическое трудоустройство:** Профессии (ГСР, ГО) не выбираются в лобби, а выдаются игровыми персонажами или через терминалы в процессе игры.
* **Очки Лояльности (LP):** Социальный лифт Сити-17. Зарабатывайте LP через труд и доносы, чтобы получить доступ к привилегиям и защите Альянса.
* **Система «Око Альянса»:** Автоматические рамки-сканеры на КПП логируют все передвижения граждан для Диспетчера.
* **Производственный цикл:** Полноценный завод рационов, где ГСР перерабатывает биомассу в пищевые брикеты для всего города.
* **Живой мир:** Система Ghost-ролей позволяет погибшим игрокам возвращаться в игру в виде хедкрабов и зомби по мере заражения секторов.

## ⚙️ Сборка и запуск

Проект требует установленного Python и .NET SDK.

1.  **Клонируйте репозиторий:**
    ```bash
    git clone https://github.com/Station17/station-17-public.git
    ```
2.  **Обновите подмодули:**
    ```bash
    git submodule update --init --recursive
    ```
    3.  **Скомпилируйте проект:**
    ```bash
    dotnet build
    ```
